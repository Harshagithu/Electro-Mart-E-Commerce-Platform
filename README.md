# ElectroMart

Electronics shopping platform built with React, Redux Toolkit, Context API, Axios and JSON Server.

## Setup
1. `npm install`
2. Terminal 1: `npm run server` (starts JSON Server on port 5000)
3. Terminal 2: `npm run dev` (starts Vite dev server)
4. Demo admin login: admin@electromart.com / admin123
5. Demo customer login: customer@electromart.com / customer123

## Tech Stack
React JS, React Router DOM, Redux Toolkit, Context API (Auth), Axios, JSON Server, Bootstrap

## Folder Structure
See src/ for components, pages, admin, redux, context, services, routes.
